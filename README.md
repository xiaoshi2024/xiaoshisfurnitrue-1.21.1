# 🏠 小噬的家具 - 让生活更有温度！

厌倦了原版单调的方块？试试这 8 款会动的家具吧！

## ✨ 特色功能

- ✅ **饮水机** - 接水接药水，还能装感染水！
- ✅ **洗衣机 + 搓衣板** - 衣物清洁两件套
- ✅ **晾衣机** - 衣服晾干还能修装备
- ✅ **燃气灶 + 燃气罐** - 做饭有加成
- ✅ **抽油烟机** - 厨房油烟全吸走
- ✅ **微波炉** - "叮"一下就能吃

## 🎬 丝滑动画

所有家具都有 **GeckoLib** 驱动动画，每一个动作都丝滑流畅。

## 🎮 支持版本

现已支持 **Minecraft 1.21.1**，快来打造你的温馨小窝吧！

---

## 📦 安装

1. 安装 NeoForge Mod 加载器（对应 1.21.1 版本）
2. 安装 GeckoLib 依赖模组
3. 将本模组 Jar 放入 `.minecraft/mods` 文件夹
4. 启动游戏，享受吧！

## 🔧 开发环境

```bash
# 刷新依赖
./gradlew --refresh-dependencies

# 清理构建
./gradlew clean

# 构建 Jar
./gradlew build
```
